Installing for development
----------------------------

Clone:

.. code-block:: bash

  git clone https://github.com/crouffer/repertoire.git


Install in development mode, including test dependencies:

.. code-block:: bash

  cd repertoire
  pip install pipenv --user
  pipenv install --dev


Running the tests
-----------------

To run the tests use:

.. code-block:: bash

  cd repertoire
  pytest
